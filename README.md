<h1>🚀 Scholarship Hunter</h1>

<p>
An AI-powered platform that helps students discover and apply for scholarships tailored to their profile.
</p>

<hr>

<h2>🎯 Problem Statement</h2>
<p>
Students often miss scholarships due to lack of awareness, scattered information across platforms, and complex application processes.
</p>
<hr>

<h2>💡 Proposed Solution</h2>

<h3>🟥 Centralized Platform & Profile Management</h3>
<ul>
  <li>A centralized web platform where students can register and create their academic profiles.</li>
  <li>Enables structured storage of academic and personal data for better scholarship matching.</li>
</ul>

<h3>🟥 AI-Based Recommendation & Assistance</h3>
<ul>
  <li>AI-based recommendation engine that analyzes user data and suggests relevant scholarships using filtering and ranking.</li>
<li>AI assistant to generate application content and simplify complex form filling.</li>
</ul>

<h3>🟥 Personalized Dashboard & Smart Exploration</h3>
<ul>
  <li>Personalized dashboard displaying recommended scholarships, application status, and deadlines.</li>
  <li>Smart search and filtering options for manual exploration.</li>
</ul>

<h3>🟥 Community, Mentorship & Support System</h3>
<ul>
  <li>Community platform for peer discussions and guidance.</li>
  <li>Mentorship support and document assistance to improve application success.</li>
</ul>
<hr>

<h2>🧠 How It Works</h2>
<ol>
  <li>User signs up and logs in</li>
  <li>User fills profile or explores scholarships</li>
  <li>System processes data using AI recommendation engine</li>
  <li>Scholarships are filtered, scored, and ranked</li>
  <li>User receives personalized recommendations</li>
  <li>User applies with AI assistance</li>
  <li>User gets support via community and mentors</li>
</ol>

<hr>
<h2>🧠 AI Recommendation Logic</h2>
<p>The platform uses a rule-based scoring system:</p>
<ul>
  <li>+40 → Academic match</li>
  <li>+30 → Income match</li>
  <li>+20 → Location match</li>
  <li>+10 → Field match</li>
</ul>
<p>
Scholarships are ranked based on total score to provide the best matches.
</p>

<h2>📡 API Endpoints</h2>
<ul>
  <li><code>POST /api/users</code> → Create user</li>
  <li><code>GET /api/users/:id</code> → Get user</li>
  <li><code>POST /api/scholarships</code> → Add scholarship</li>
  <li><code>GET /api/scholarships</code> → Get all scholarships</li>
  <li><code>GET /api/recommend/:userId</code> → Get AI recommendations</li>
</ul>
